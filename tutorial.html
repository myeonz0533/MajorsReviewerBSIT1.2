1Ô∏è‚É£ Understanding HTML (index.html)

HTML (HyperText Markup Language) provides the structure of the webpage. Think of it as the skeleton of your application.

Key HTML Concepts Used:

  • <div> - Container elements that group content
  • <header> - Top section with title and theme toggle
  • <nav> - Navigation buttons (Reading Mode / Quiz Mode)
  • <aside> - Sidebar for topic navigation
  • <main> - Main content area
  • <button> - Clickable buttons
  • id attributes - Unique identifiers for JavaScript to find elements
  • class attributes - Groups of elements for styling
üí° Tip: HTML elements are like building blocks. Each element has a purpose and can contain other elements.

Example: Basic HTML Structure

<div class="container"> <header> <h1>Title</h1> </header> <div id="content"> <p>Content goes here</p> </div> </div>

2Ô∏è‚É£ Understanding CSS (styles.css)

CSS (Cascading Style Sheets) controls the appearance of your webpage. It's like the paint and decoration for your HTML structure.

Key CSS Concepts Used:

  • Selectors - Target HTML elements (e.g., .class-name, #id-name)
  • Properties - What you're changing (e.g., color, background, padding)
  • Values - The actual change (e.g., red, 20px, #7B2CBF)
  • CSS Variables - Reusable values (e.g., --primary-purple: #7B2CBF)
  • Media Queries - Responsive design for mobile devices
  • Flexbox - Layout system for arranging elements

Example: CSS Styling

/* Target an element by class */ .button { background: #7B2CBF; /* Purple background */ color: white; /* White text */ padding: 10px 20px; /* Space inside */ border-radius: 5px; /* Rounded corners */ } /* Target an element by ID */ #header { text-align: center; margin-bottom: 20px; } /* Hover effect */ .button:hover { background: #5A189A; /* Darker purple on hover */ }
‚ú® Pro Tip: CSS Variables (like --primary-purple) allow you to change colors throughout your entire app by updating just one value!

Dark Mode Implementation

The app uses CSS variables and a data-theme attribute to switch between light and dark modes:

/* Light mode colors */ :root { --primary-color: #7B2CBF; --bg-color: #FFFFFF; } /* Dark mode colors */ [data-theme="dark"] { --primary-color: #6BA83A; --bg-color: #1A1A1A; }

3Ô∏è‚É£ Understanding JavaScript (app.js)

JavaScript adds interactivity to your webpage. It makes buttons work, handles user clicks, and updates content dynamically.

Key JavaScript Concepts Used:

  • Variables - Store data (let currentTopic = 0)
  • Functions - Reusable blocks of code
  • Event Listeners - React to user actions (clicks, etc.)
  • DOM Manipulation - Change HTML content dynamically
  • Arrays - Lists of data
  • Objects - Collections of related data
  • Conditionals - Make decisions (if/else)
  • Loops - Repeat actions (forEach)

Example: Basic JavaScript Function

// Function to change topic function loadTopic(index) { // Get the topic data const topic = topics[index]; // Find the HTML element and update it document.getElementById('topic-title').textContent = topic.title; document.getElementById('topic-content').innerHTML = topic.content; } // Event listener - runs when button is clicked document.getElementById('next-btn').addEventListener('click', function() { currentTopic++; loadTopic(currentTopic); });

How Event Listeners Work

Event listeners "listen" for user actions and respond:

// When user clicks a button button.addEventListener('click', function() { // This code runs when button is clicked alert('Button was clicked!'); });
